# TextTuner

Инструмент для адаптации текста под целевой стиль с лингвистическим анализом. 
Проект разработан студентом-лингвистом для анализа и улучшения стилистических характеристик текста.

## Возможности

- **Анализ стилистических характеристик** текста
- **Адаптация под целевой стиль** (научный, художественный, официально-деловой, разговорный)
- **Рекомендации по улучшению** стиля с конкретными примерами
- **Статистика стилистического соответствия** с визуализацией отклонений
- **Лингвистический анализ** с использованием pymorphy3 для морфологического разбора

## Поддерживаемые стили

| Стиль | Описание | Характеристики |
|-------|----------|----------------|
| **Научный** | Точность, формальность, объективность | Высокая формальность, терминология, сложные конструкции |
| **Художественный** | Выразительность, эмоциональность, образность | Метафоры, эпитеты, разнообразная лексика |
| **Официально-деловой** | Стандартизированность, точность, безличность | Клише, штампы, канцеляризмы |
| **Разговорный** | Неформальность, простота, естественность | Простые конструкции, разговорная лексика |

### Установка

1. Клонируйте репозиторий:
```bash
git clone https://github.com/puglyann/TextTuner.git
cd text-tuner
```

2. Создайте и активируйте виртуальное окружение:
```bash
python -m venv .venv
source .venv/bin/activate  # Linux/Mac
.venv\Scripts\activate  # Windows
```

3. Установите зависимости:

```bash
pip install -e
```
